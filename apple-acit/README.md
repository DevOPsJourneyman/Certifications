# Apple Certified IT Professional

**Exams:** SUP-2026 (Apple Certified Support Professional) + DEP-2026 (Apple Certified IT Professional — Deployment and Management)
**Provider:** Apple
**Target:** SUP-2026 flexible (inside the 7-week plan, after Week 4) · DEP-2026 in Week 9

## Exam Breakdown

| Exam | Objectives |
|------|-----------|
| SUP-2026 — Device Support | Hardware, startup/recovery, diagnostics, FileVault, T2/Secure Enclave |
| DEP-2026 — Deployment & Management | ABM, ADE, supervision, enrollment types, config profiles, DDM, VPP |

## Lab environment

Kambi manages Macs and iPads via Microsoft Intune (not Jamf) with an active ABM/DEP enrollment requirement — this is the real lab for DEP-2026 content, used read-only. Personal test hardware: 2× Apple Silicon MacBook Pro; one is a spare already assigned in Kambi's ABM, awaiting deployment, used for SUP-2026 hands-on labs and to observe the live ADE flow when it enrolls.
